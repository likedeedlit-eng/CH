## 一致性哈希算法说明

### 算法原理

一致性哈希算法通过将机器和数据映射到一个虚拟的哈希环上，实现数据的分布式存储。当添加或删除机器时，只有少量数据需要重新分配，大大减少了数据迁移的开销。

### 核心步骤

1. **哈希计算**：使用`splitmix64`算法计算机器和数据的哈希值
2. **数据分配**：数据根据哈希值分配到环上最近的机器节点
3. **节点管理**：支持动态添加和删除机器节点
4. **数据迁移**：节点变化时自动进行最小化的数据迁移

### 哈希值范围

- 使用31位整数范围：`0` 到 `2147483647`
- 确保良好的分布均匀性和较低的哈希碰撞概率
